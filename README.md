# HTML-practice

## 연습문제1
```html
<!DOCTYPE html>
<html <head>
<meta charset="utf-8">
<title></title>
</head>

<body>
  <h1>Html5 웹 관련 업무를 위해서는 필수</h1>
  <h3>코딩... 어려운 것이 아니라 적응이 안된 것이다.</h3>
  <p>

    웹퍼블리셔,웹기획 웹개발자가 되기 위해서는 기본적인 HTML을 알아야 합니다. HTML은 웹직무의 기본이자 필수 입니다.

    <p>
      처음 코딩을 공부하면 어렵다고 생각할 수 있습니다. 누구나 새로운 규칙을 공부하는 것은 쉽지 않았습니다.
      하지만 코딩은 잘하는 것이 아니라 익숙해지는 것 입니다. 지금의 숙련자들도 연습과 시행착오가 있었습니다.
    </p>

    <blockquote>
      <hr>
      지금은 여러 브라우저와 기기의 다양화로 인하여 표준화된 html이 필요합니다. html5는 표준화된 태그언어 입니다.
    </blockquote>
  </p>
</body>

</html>
```
## 연습문제2
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title></title>
  </head>
  <body>
    <h2>시작이 반이다.</h2>
     <p>
       우리는 무슨 일을 시작하기 전에 <em>이 일을 잘 할 수 있을까?</em> 라는 막연한 불확실성을 가지고 있습니다.
       누구에게나 처음 하는 일은 두려움을 갖기 마련입니다. 중요한 것은 일이나 학습을 하다보면
       <strong><em>'실수를 할 수 있다. 실수를 통해 무엇이 잘못되었는지 기억하자'</em></strong>
       라는 것입니다 현명한 사람도 실수를 합니다. 다만 반복을 하지는 않습니다.
     </p>
  </body>
</html>
```
